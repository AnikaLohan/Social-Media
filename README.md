# Social-Media
Developed a full-stack social media platform that allows users to create accounts, share media content (images, videos, audio), connect with other users via friend requests, and securely manage authentication including password reset functionality. The application focuses on scalability, secure user management, and real-world social networking features.

MongoDB | Express | React | Node.js
Developed a full-stack social media application enabling users to share images, videos, and audio content.
Implemented user authentication, friend request functionality, and secure password reset workflow.
Built a responsive frontend using React and Tailwind CSS with global state management via Redux Toolkit.
Designed RESTful APIs using Node.js and Express, with JWT-based authentication.
Used MongoDB for scalable data storage and relationship handling.
Managed the complete codebase using GitHub, following proper push/pull and version control practices.
